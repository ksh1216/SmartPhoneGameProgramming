##### Activity 구성
MainActivity - 기본적인 시작지점 현재는 시작 버튼 하나로 간단하게만 구성되어있음 버튼을 클릭시 SuikaActivity를 호출하는 역할
SuikaActivity - 가상 화면을 만들고 MainScene을 반환


##### Scene 구성
현재는 단일 신으로 이루어져있음
BACKGROUND, FRUIT, TOP_FRUIT, UI, COUNT 레이어를 만들어 관리중


##### MainScene 등장 오브젝트 - Fruit
###### Class 구성정보
아직까지는 비트맵 이미지 없이 Canvas에 직접 도형을 그려 게임 오브젝트를 생성
그림구성 - draw() 함수를 통해 Paint 객체를 사용하여 크기에 따라 지정된 색을 채운 후 렌더링
동작구성 - 내부적으로 수평속도, 수직속도, 중력, 마찰력 변수를 통해 과일을 떨어뜨리고 부딪히는 물리 낙하. 위 계산식과 물리적 밀어내기의 경우 AI의 도움을 통해 식을 얻었음
상호작용 정보 - 과일끼리의 충돌
두 객체의 중심점 거리와 반지름의 합을 비교하여 충돌을 판정
다른 과일과 부딪힐 시 서로 밀어내고 굴러가는 연산이 진행됨
같은 과일끼리 부딪힐 시 두 객체를 삭제하고 가운데에 그 다음 단계의 과일을 생성
###### 핵심 코드 설명
getGradeColor(grade: Int) - 각 과일의 색상을 결정
update(gctx: GameContext) - 외부에서 조작되는 상태 변수에 중력과 마찰력을 연산하여 최종적인 과일의 좌표를 확정시킴
