# SmartPhoneGameProgramming

## 프로젝트 수박게임

### 게임컨셉
닌텐도 e숍에서 판매 및 플레이 가능하던 Suika Game(수박게임)을 모티브로 제작해볼 예정

핵심 플레이 - 작은 과일들을 합쳐서 큰 과일을 만들어 높은 점수를 얻는 퍼즐 게임

핵심 메카닉 - 낙하 및 물리 연산, 합성 시스템, 최고 점수 갱신

### 개발범위
과일 종류 - 6개까지 제작 예정 (가장 큰 과일은 수박으로 고정)

물리 연산 구현 - 과일 낙하, 다른 과일 끼리의 충돌

점수 관리 - 최고 점수 저장

### 개발 일정
1주차 : 기존 FighterPlane의 Game framework를 분석 및 사용

2주차 : 터치 시 과일 생성 및 낙하 구현 (중력 구현)

3주차 : 과일 간 충돌 처리 (1) - 바닥 및 다른 과일 끼리의 충돌

4주차 : 과일 간 충돌 처리 (2) - 같은 과일 끼리의 합성

5주차 : 과일 간 충돌 처리 (3) - 충돌 관련 수정 및 버그 픽스

6주차 : 데드 라인 구현 및 점수 계산, 기록 시스템

7주차 : UI 최적화 (메인 메뉴) 및 지연 기능 구현

8주차 : 버그 수정 및 빌드, 배포
